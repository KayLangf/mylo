## Notes on Knowledge Base Coverage

This case study's knowledge base includes full content for:
- **FNS 360** (Determining Benefit Levels) — both the outdated (2021) and current (2025) versions, used deliberately to test source-authority conflict handling
- **FNS 227** (Non-citizen Requirements) — via the SNAP Noncitizen Eligibility program page and FNS-CN-01-2026 change notice
- **FNS 260** (ABAWD) — via the ABAWD Work Requirements program page

Sections not included (e.g., FNS 210 Household Composition, FNS 300 Sources of Income, FNS 425 Expedited Services) represent real gaps in this POC's knowledge base. If the agent is asked about topics governed by these sections, it should honestly state it doesn't have that information rather than guessing — this is an intentional test of the agent's groundedness and honesty guardrails given the deliberately narrow scope of a 20-hour build.
